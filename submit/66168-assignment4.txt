It is difficult for teacher to manage student’s assignment (upload assignment, submit assignment, marking, recording of marks, checking of plagiarism) especially when number of students is large, so that this project will help teachers to manage student’s assignment easily 
This is the system which manage the student’s assignment in which allow teacher to upload the assignment, marking assignment and automatic will provide student result also while student may submit the assignment, so the system will check the deadline and block other assignment then start checking for assignment plagiarism.
